# Custom Test Images

Place your test images here for model testing.

## Supported Formats
- JPG/JPEG
- PNG

## How to Use

1. Copy your test images to this folder
2. Run the testing script:
   ```
   python test_custom_images.py
   ```
3. Results will be saved in `test_results/` folder
